Para acceder a Oracle añadimos la URL de conexion, el usuario y la contraseña con la que queremos conectarnos, luego mediante un try catch para controlar errores nos conectamos mediante la clase Connection utilizando DriverManager.getConnection, para que el codigo funcione habra que añadir el driver JDBC de Oracle: "ojdbc17.jar". Hay que tener en cuenta que se puede realizar de otra forma mediante Maven en donde se añadira todo actualizado automaticamente
